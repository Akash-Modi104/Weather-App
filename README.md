
# Weather App

This is web based App where user can reigster themself and login and get the current details of weather of any part of world by simple search


## Technology Stack

- Backend :- 
    python, flask , sqlalchemy , marshmallow, requests
  
- Frontend :- Angular
- Database :- sqlite
